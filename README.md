## 制定资源
Flutter 使用 pubspec.yaml 文件，位于项目根目录，来识别应用程序所需的资源。
1、指定文件
flutter:
  assets:
    - assets/my_icon.png
    - assets/background.png

2、如果要包含一个目录下的所有 assets，需要在目录名称的结尾加上 /：
flutter:
  assets:
    - directory/
    - directory/subdirectory/
